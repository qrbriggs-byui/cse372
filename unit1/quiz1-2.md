# Quiz 1-2: Understanding Business, User, Functional, and Non-Functional Requirements

## Question 1
**What do business requirements primarily explain?**
- A. How the software interface will look
- B. Why the project exists and what outcomes it aims to achieve
- C. The exact features to be built
- D. Who the target users are  
✅ **Correct Answer:** B

---

## Question 2
**Which of the following is a good example of a business requirement?**
- A. The system shall send a confirmation email
- B. Pages should load within two seconds
- C. Reduce customer service calls by 30%
- D. Users can reset their password  
✅ **Correct Answer:** C

---

## Question 3
**User requirements focus on which aspect of the project?**
- A. The internal database structure
- B. The needs and goals of the people who use the system
- C. The organization's financial metrics
- D. The system's uptime and performance  
✅ **Correct Answer:** B

---

## Question 4
**Which of the following is a valid user requirement example?**
- A. As a customer, I want to track my delivery status so I can plan my day
- B. The system must pass OWASP security testing
- C. The system shall log errors to a file
- D. System availability must be 99.9%  
✅ **Correct Answer:** A

---

## Question 5
**Functional requirements describe:**  
- A. What the system must do
- B. Why the project is being developed
- C. The emotional needs of the user
- D. How well the system must perform  
✅ **Correct Answer:** A

---

## Question 6
**Which statement best fits as a functional requirement?**
- A. The system must be available 99.9% of the time
- B. Users can reset their password via a secure link
- C. The interface must meet WCAG accessibility standards
- D. The website should load quickly on mobile devices  
✅ **Correct Answer:** B

---

## Question 7
**What do non-functional requirements (NFRs) define?**
- A. What features need to be implemented
- B. The qualities of the system such as performance and usability
- C. The exact test cases to use
- D. The programming language chosen  
✅ **Correct Answer:** B

---

## Question 8
**Which of the following is a non-functional requirement?**
- A. The system shall send a confirmation email after registration
- B. The system must allow admin login
- C. Pages should load within two seconds on a 4G connection
- D. Users can update their profile information  
✅ **Correct Answer:** C

---

## Question 9
**What happened to the team that ignored performance requirements in their mobile ordering app?**
- A. The app performed flawlessly under all conditions
- B. The app looked outdated but worked well
- C. The app crashed during high traffic times
- D. The app was too simple for user needs  
✅ **Correct Answer:** C

---

## Question 10
**How do the four types of requirements fit together?**
- A. They are unrelated and should be developed independently
- B. Business defines the goal, user defines needs, functional defines actions, non-functional defines quality
- C. User and non-functional requirements are the same
- D. Only functional requirements matter for success  
✅ **Correct Answer:** B
